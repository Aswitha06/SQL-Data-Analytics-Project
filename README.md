# 📊 Data Analytics SQL Project 

## 📌 Project Overview

This project demonstrates SQL skills by building a Sales Analytics database from scratch. It covers database design, data manipulation, data analysis, and advanced SQL concepts used in Data Analytics.

The project simulates a real-world sales environment involving customers, employees, departments, orders, and products.

## 🎯 Objectives

* ✅ Design and create a relational database
* ✅ Perform data insertion, updates, and deletion operations
* ✅ Analyze business data using SQL queries
* ✅ Generate insights using aggregate functions
* ✅ Work with joins and subqueries
* ✅ Implement views, indexes, stored procedures, and functions

## 🗂️ Database Tables

* 🏢 Departments
* 👨‍💼 Employees
* 👥 Customers
* 🛒 Orders
* 🛍️ Products
* 📦 Order Items
